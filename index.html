🖥 Chapter 1: ကွန်ပျူတာဆိုတာ ဘာလဲ?

What is a Computer? — သူ့ရဲ့ အလုပ်လုပ်ပုံ

ကွန်ပျူတာ = Electronic Brain!

ကွန်ပျူတာဆိုတာ ဂိမ်းဆော့ဖို့ ဒါမှမဟုတ် စာရိုက်ဖို့သက်သက် မဟုတ်ဘူး — သူဟာ အချက်အလက်တွေကို တစ်စက္ကန့်မှာ billion ချီပြီး တွက်ချက်နိုင်တဲ့ Electronic Device တစ်ခုပါ!

1
Electricity ဝင်လာတယ် — Power button နှိပ်လိုက်တာနဲ့ Electric current တွေ Circuit ထဲ ဝင်ကာ CPU, RAM, Storage တွေ စသက်ဝင်လာတယ်
2
BIOS/UEFI ပြေးတယ် — Motherboard ရဲ့ chip ထဲမှာ BIOS ဆိုတဲ့ tiny program ရှိပြီး hardware အကုန် စစ်ဆေးပေးတယ်
3
OS Load ဖြစ်တယ် — Storage (SSD/HDD) ထဲက Windows ကို RAM ထဲ ကူးတင်တယ်
4
0 နဲ့ 1 တွက်တယ် — CPU မှာ transistor သန်းချီ ပါပြီး ON/OFF (1/0) ဖြင့် Calculation ထုတ်ပေးပါတယ်
5
မင်း မြင်ရတယ် — ရလဒ်ကို GPU → Monitor → မင်းရဲ့မျက်စိ ဆီ ပေးပို့တယ်!
🤩
Did you know? မင်းရဲ့ CPU ဟာ တစ်စက္ကန့်မှာ 3,000,000,000 ကြိမ် (3 GHz) လုပ်ဆောင်နိုင်တယ်! 🚀

🧠 CPU — ကွန်ပျူတာရဲ့ ဦးနှောက်

CPU (Central Processing Unit) ဆိုတာ ကွန်ပျူတာရဲ့ ဦးနှောက်ပါ — Input တွေကို ယူပြီး တွက်ချက်ကာ Output ထုတ်ပေးပါတယ်။

1
Fetch — RAM ထဲကနေ next instruction ကို ဆွဲထုတ်တယ်
2
Decode — Instruction ကို 0/1 binary signal တွေ အဖြစ် ပြောင်းတယ်
3
Execute — ALU က တွက်ချက်မှုကို လုပ်တယ် (+, -, ×, ÷)
4
Store — ရလဒ်ကို Register ဒါမှမဟုတ် RAM ထဲ ထည့်ထားတယ်

💾 Storage — SSD vs HDD

⚡
SSD (Solid State Drive)
• Flash Memory Chip သုံး
• ပိုမြန်တယ် (10x faster)
• Moving parts မပါ
• ဈေးနှုန်း ပိုကြီး
Data ကို NAND Flash Memory chip တွေထဲမှာ electrical charge အဖြစ် သိမ်းတယ်။ Moving parts မပါတာကြောင့် အမြန်ဆုံး access လုပ်နိုင်ပြီး ခံနိုင်ရည်လည်း ကောင်းပါတယ်!
🗄️
HDD (Hard Disk Drive)
• Magnetic Disk သုံး
• နှေးကွေး
• Disk ပတ်ရသော Motor ပါ
• ဈေးနှုန်း ချိုသာ
Magnetic coating ပါတဲ့ spinning disk ပေါ်မှာ Read/Write head က ရွှေ့ကာ data ဖတ်/ရေးတယ်!
💡
Speed! SSD က Windows ကို 8-10 seconds မှာ ဖွင့်နိုင်ပြီး HDD က 30-60 seconds ကြာနိုင်ပါတယ်!

💾 Chapter 3: Software

The Digital Brain — မမြင်ရသော မရှိမဖြစ်သော!

Software ဆိုတာ ဘာလဲ?

Software ကို မျက်စိနဲ့ မြင်မရ၊ လက်နဲ့ မကိုင်ရ — ဒါပေမဲ့ Software မရှိရင် Hardware က သံတိုသံစပါပဲ! Software ဟာ ကွန်ပျူတာကို "ဘာလုပ်ရမလဲ" ဆိုပြီး ပြောပေးတဲ့ Instructions တွေပါ။

1
Program ကို ဖွင့်တယ် — Storage ထဲက program file တွေကို RAM ထဲ ကူးတင်တယ်
2
CPU ကို ညွှန်ကြားတယ် — Instructions (code) တွေကို CPU ဆီ တစ်ကြောင်းချင်း ပို့တယ်
3
Result ပြပေးတယ် — CPU ရဲ့ ရလဒ်ကို Screen, Speaker, Printer ဆီ ပြန်ပို့ပြီး မြင်ရ/ကြားရတယ်
